# Customer Shopping Behavior Analysis

## Overview

An end-to-end data analytics project analyzing **3,900 customer transactions** to uncover shopping patterns, customer segments, product preferences, and revenue trends. The project combines **Python, PostgreSQL, and Power BI** to transform raw data into actionable business insights.

## Tools & Technologies

* Python (Pandas, NumPy)
* PostgreSQL
* SQL
* Power BI

## Project Workflow

1. Data Cleaning & Preprocessing using Python
2. Feature Engineering and Missing Value Handling
3. SQL-Based Business Analysis in PostgreSQL
4. Interactive Dashboard Creation in Power BI
5. Business Insights & Recommendations

## Key Analyses

* Revenue by Gender
* Subscribers vs Non-Subscribers
* Customer Segmentation (New, Returning, Loyal)
* Top-Rated Products
* Discount Impact Analysis
* Revenue by Age Group
* Shipping Type Performance

## Key Insights

* Subscribers generated higher revenue.
* Loyal customers contributed significantly to sales.
* Some products relied heavily on discounts.
* Customer spending varied across age groups and purchase behavior.
